• HTML

    HTML은 웹 페이지의 뼈대를 구성하는 마크업 언어 입니다.

    표현하고자 하는 컨텐츠의 목차를 구성하고 내용을 담는 역할을 합니다.

  • CSS

    CSS는 준비한 HTML문서에 디자인을 적용하는 언어 입니다.

    목차를 메뉴로 표현하고 내용을 컨텐츠로 구성하여 약간의 다이나믹한 효과를 부여하는 역할을 합니다.

  • Javascript

    Javascript는 HTML과 CSS로 구성된 웹 페이지에 생명을 불어 넣습니다.

    사용자와 대화식 웹 페이지를 구성할 수 있고 다채로운 애니메이션 효과도 적용할 수 있는 강력한 언어 입니다.

    Javascript를 사용하여 우리는 사용자 경험을 구현할 수 있습니다.

  • jQuery

    jQuery는 세상에서 가장 많이 사용되는 Javascript 라이브러리 입니다.

    간결한 표현방법과 다양한 플러그인이 여러분들의 웹 페이지에 다채로운 사용자 경험을 제공할 것 입니다.
